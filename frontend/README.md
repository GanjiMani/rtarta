# RTA Frontend - Tailwind UI (Complete)

This project is a full React + Vite + Tailwind UI skeleton for the RTA Investor & Admin portals.

Features:
- Tailwind CSS (utilities only)
- Admin sidebar layout + Investor top-nav layout
- Recharts integrated for charts (dummy data)
- Mock Auth context + Protected routes
- All pages from the RTA screens implemented with responsive UI

Run:
1. npm install
2. npm run dev
3. Open http://localhost:5173

Note: Replace mockApi in src/services/mockApi.js with real backend API calls.
